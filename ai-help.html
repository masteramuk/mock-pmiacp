Recommended Workflow

Good workflow
1. Export the current dataset JSON from the app or dataset tools page.
2. Ask the AI to edit or add questions using the exact schema.
3. Import the JSON or CSV into Dataset Tools, or paste copied text for review.
4. Edit the staged questions before merging and downloading the merged dataset.

Prompt Template

You are helping maintain a PMI-ACP mock exam dataset.

Requirements:
- Follow the PMI-ACP Examination Content Outline (October 2024)
- Domains: Mindset, Leadership, Product, Delivery
- Scenario-based questions only
- 4 options (A-D)
- Use single-answer and multiple-answer questions where appropriate
- Include explanation
- No duplicates
- Keep IDs stable if editing existing questions
- Return valid JSON items only

Schema:
{
  "id": 1,
  "question": "string",
  "options": { "A": "string", "B": "string", "C": "string", "D": "string" },
  "answer": "A" or ["A","C"],
  "explanation": "string",
  "domain": "Mindset",
  "difficulty": "medium",
  "selectionType": "single"
}

Task:
- Improve wording and distractors for the supplied questions
- Or add new questions without changing the schema
